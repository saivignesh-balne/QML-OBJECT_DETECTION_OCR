This project uses a hybrid quantum design rather than a pure quantum image-processing pipeline. That decision is important. Raw images are too high-dimensional for direct practical quantum handling in the current near-term setting. A hybrid design is more realistic: preprocessing and feature extraction remain classical, while the classification stage becomes the point where quantum models can be introduced and evaluated fairly.

Voiceover:

To support quantum classification, the project implements two feature-encoding methods. In angle encoding, classical feature values are mapped to quantum rotation angles. In amplitude encoding, classical data is normalized and embedded into the amplitudes of a quantum state. These encodings allow classical image-derived features to be transformed into quantum-compatible representations before model evaluation.

If text is not clearly available, the system still reports the predicted object and marks the text output accordingly.

SHORT READY-TO-RECORD VERSION
==================================================

This project presents a hybrid quantum machine learning pipeline for object detection, classification, and OCR. It uses chip packets, medicine boxes, and bottles as the current benchmark objects while comparing classical and quantum-oriented models within the same practical workflow.

The pipeline begins with image preprocessing using OpenCV for noise reduction, contrast enhancement, and binarization. A YOLO-based detector then identifies the object location, and the detected region is extracted as the region of interest. That ROI is passed into two branches: a classification branch and an OCR branch.

In the classification branch, classical image features are extracted and then evaluated using both classical and hybrid quantum models. The classical group includes SVM, Logistic Regression, Random Forest, and MLP. The quantum group includes QSVM variants and variational quantum classifiers implemented with Qiskit. To support quantum processing, the project uses both angle encoding and amplitude encoding to transform classical features into quantum-compatible representations.

In parallel, the OCR branch uses Tesseract and TrOCR to extract visible printed text from the object surface. The final output shows the predicted object label and the extracted text.

The project also includes a full user interface for training, benchmarking, and inference. This allows models to be saved, reused, compared, and demonstrated visually.

The benchmark results show that classical models remain the strongest and most stable production choice, while at least one hybrid quantum model, the amplitude-encoding QSVM, achieves competitive performance on the ROI benchmark. This makes the project valuable not only as a working application, but also as a realistic research platform for evaluating the strengths and limitations of QML in practice.
